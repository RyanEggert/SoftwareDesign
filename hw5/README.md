Software Design: Game Development
======================

Game development assignment for Software Design Fall 2014. [This](https://github.com/RyanEggert/SoftwareDesign/tree/master/hw5 "Go to the /hw5 directory") directory contains all of the final code. The only external package required is Pygame.

**To play the game** and start from the welcome screen, please run `welcome.py`. No parameters are necessary.

To see development history/commits, please refer to the development repository [here](https://github.com/RyanEggert/SoftwareDesign_GameDev "Go to the development repository")
