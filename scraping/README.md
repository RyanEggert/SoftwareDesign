# esca.py

The final code can be found in the "escapy" folder. API keys for [Google QPX Express API][1] and [forecast.io][2] are neccessary to run this program.
Optional is an API key<sup>*</sup> to use the [GoogleV3 geocoder API][3]. Scripts outside of the "escapy" folder were used for testing and experimentation purposes.


<sup>*</sup>Because Google QPX Express and the Google geocoder are both operated by Google, the same API key can work for both. Howevever, access to both must be enabled from within the "Google Developers Console".

[1] https://developers.google.com/qpx-express/                      "Google QPX Express API"
[2] https://developer.forecast.io/                                  "forecast.io"
[3] https://developers.google.com/maps/documentation/geocoding/     "GoogleV3 geocoder API"
